# LaptopShop
Задание: Реализовать web-приложение магазин ноутбуков. В базе данных должны храниться данные о товарах. Сайт должен иметь 2 страницы:
1. Страница с формой входа на сайт.
2. Страница магазина ноутбуков.
На странице для USER должна быть возможность просматривать список ноутбуков и кнопка "Купить". При нажатии на кнопку, количество ноутбуков должно уменьшиться на 1.
На странице для ADMIN должна быть возможсть просматривать список ноутбуков, добавить +1 ноутбук, удалить -1 ноутбук и форма для загрузки картинки. В базе данных хранить название или ссылку на файл.
